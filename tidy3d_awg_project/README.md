# Tidy3D AWG Project

This project directory is organized so the main deliverables are easier to find:

- `docs/`: project notes, summaries, and milestone writeups
- `scripts/`: reproducible Python workflows for analysis and report generation
- `results/tables/`: CSV outputs and numeric summaries
- `results/figures/`: plots, HTML report previews, and exported figures
- `reports/`: final `.docx` report deliverables
- `data/high_fidelity_awg/`: downloaded Tidy3D high-fidelity simulation data

The notebooks remain at the project root because they are part of the original interactive workflow.
