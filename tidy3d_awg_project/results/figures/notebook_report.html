1. Report Objective

This document consolidates the main results from the project notebooks into a homework-style report. The goal is to show how the baseline silicon strip waveguide MODE study was extended into a width sweep and then translated into a first-pass 4-channel C-band AWG design.

2. Notebook Scope and Workflow

  • waveguide_mode.ipynb established a baseline MODE solution for a 500 nm x 220 nm silicon strip waveguide in SiO2.
  • waveguide_width_sweep.ipynb scanned widths from 0.40 um to 0.60 um at 1550 nm to extract n_eff and n_group.
  • awg_initial_design.ipynb converted the width-sweep result into channel targets, FSR, diffraction order, and array-length estimates for a simplified AWG.

3. Baseline MODE Result

The baseline notebook confirms that the selected silicon strip waveguide supports a fundamental guided mode near 1550 nm. The 0.50 um width case is used as the reference design point because it is already represented in the initial MODE run and sits near the middle of the width sweep range.

4. Width Sweep Results

A low-cost MODE sweep was run at 1550 nm for widths from 0.40 um to 0.60 um. The results show a monotonic increase in effective index and a gradual decrease in group index as the waveguide becomes wider.

Based on these trends, 0.50 um is a practical first-pass choice. It matches the earlier baseline result, avoids the extremes of the scan range, and provides n_eff = 2.4446 and n_group = 4.0512 for subsequent AWG calculations.

7. Conclusions and Next Steps

  • The notebook sequence provides a clean path from single-waveguide MODE analysis to AWG parameter estimation.
  • The 0.50 um waveguide width remains the recommended baseline because it balances continuity, moderate index values, and reportability.
  • The first-pass AWG target is a 4-channel C-band demultiplexer with channel centers at 1547.6, 1549.2, 1550.8, and 1552.4 nm.
